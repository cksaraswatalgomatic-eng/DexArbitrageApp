Liquidity Monitoring

Real-time liquidity and price data for selected cryptocurrencies. Line chart shows price in USDT, bars show liquidity (volume per 2-minute interval) in USDT. Liquidity bars are filtered to exclude outliers using IQR method (Q3 + 2*IQR) by default, or set a custom maximum value below.
